這是我的個人作品集網站，展示了身為前端開發者的技術實力與專案成果。網站採用 **React 18** 打造，具備響應式設計與流暢的動態互動效果。

## ✨ 功能特點

- **打字機動效 (Typewriter)**：首頁具備動態文字輪播，增加視覺趣味性。
- **深淺色模式 (Dark/Light Mode)**：支援一鍵切換主題，並適配所有元件樣式。
- **平滑捲動 (Smooth Scroll)**：頂部導覽列與 CTA 按鈕具備流暢的區塊跳轉功能。
- **滾動偵測動畫 (Scroll Reveal)**：利用 `Intersection Observer` 實現區塊淡入效果。
- **響應式佈局**：使用 CSS Grid 與 `clamp()` 函式，完美適配手機與桌面螢幕。

## 🛠️ 技術棧

- **框架**：[React 18](https://react.dev)
- **建置工具**：[Vite](https://vitejs.dev)
- **樣式**：CSS-in-JS (Inline Styles) & 原生 CSS
- **動畫**：CSS Keyframes & Intersection Observer API
- **部署**：GitHub Pages / Vercel

## 📂 專案結構

- `src/components/Hero.jsx` - 首頁打字機與主要視覺區塊
- `src/components/About.jsx` - 個人簡介與學經歷
- `src/components/Skills.jsx` - 技術清單與分組展示
- `src/components/Projects.jsx` - 精選專案卡片網格
- `src/components/Contact.jsx` - 聯絡資訊與社群連結
- `src/components/Navbar.jsx` - 全域導航與主題切換
